#  Todo List App

A clean and simple to-do list web app that lets you manage your daily tasks right in the browser.

##  Features

- Add new tasks quickly
- Mark tasks as complete or incomplete
- Delete tasks you no longer need
- Clean and minimal interface

##  Built With

- HTML
- CSS
- JavaScript (Vanilla)

##  How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Shajjia/todo-list.git
   ```
2. Open the project folder
3. Double-click `index.html` to open it in your browser

No installs or setup needed!

##  Deployment

To host it live using GitHub Pages:

1. Go to your repo on GitHub
2. Click **Settings → Pages**
3. Under **Source**, select **main** branch
4. Click **Save** — your app will be live at:
   `https://ShajjiaE.github.io/todo-list`
